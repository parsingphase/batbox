About Batbox

Batbox is an tool for displaying bat recordings tagged with Guano metadata, such as those recorded by devices from Wildlife Acoustics.

As it's open source software, you're welcome to download it from GitHub and run it on your own server or desktop, with your own recordings.

Batbox was created and is developed by Richard George, aka Parsingphase.

Using this site

There are four main views on the site:

Summary View
Displays a breakdown of species observed on each day.
Calendar View
A visual summary of the density of recordings over time.
Map View
The main display page, where you can see selected recordings in a map, and play and download them. You can select by Genus, Species or date, or use the Search view for more control.
Search View
An advanced search form where you can search by time period, location and species.

Search View

Select the time range and/or species you're interested in. Scroll and pan the map to limit the search area. You'll get instant feedback in the map view as to where results are found.

To view the search results in detail on the map view, hit the search button.
